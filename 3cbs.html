CB-Safe project

Unity

Blender

LAB university CB-Safe project was a prototype to showcase a VR training simulator about decision-making in rescue situations. It was developed with the Finnish Rescue Services and EU. My task was to create animations for firefighters and make animation state machines in Unity. Animations were first made using Blender, but it was later changed to Unity and UMotion animation editor for more precise animation controls. Some animations had separate upper and lower body animations.
