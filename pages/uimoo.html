Moon Cheese App

XD

Illustrator

This was an exercise where I designed a UI for a mobile app concept that was all about cheeses. You could find cheese news, reviews, recipes, community pictures and saved favorites.

  • 1. Login
    You can create your account or use Google, Apple or Facebook accounts. If the username is not found, it can be created in a new window.
  • 2. Home
    On the home screen you can find the most important news, recommendations, user photos, reviews and etc. The bottom bar has links to favorites, search and a profile that are easy to reach.
  • 3. Item
    More detailed information about the cheese and its statistics can be found in the item screen. Users can add the cheese to favorites, post pictures and review. There is a back button on the top left.
  • 4. Profile
    The profile box contains the user's profile information: picture, username and description. The user can edit profile, app settings and view app information.
