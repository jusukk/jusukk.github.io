Firebase

This was an exercise where I created a simple HTML5 game that uses database and display scores. Scores are stored in a Firebase Realtime Database and are used to create charts by using Chart.js library. Firebase API key is restricted to this domain and has rules for data validation and deletion.

The first chart shows the top 5 scores. The second chart shows averages and how many scores have the same value. You can toggle the chart by clicking its label.

Press SPACE to hover and try to avoid obstacles. You score when obstacles reach the other side. The game ends when you hit the obstacle, and your score is then sent to the database.

SPACE = Jump, P = Pause, R = Restart
