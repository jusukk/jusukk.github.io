Game Charts

JavaScript CSS3 / Sass HTML5 Canvas Firebase Charts.js

JavaScript and HTML5 canvas game. Scores are stored in a Firebase Realtime Database and are used to create canvas charts by using a Chart.js library.

The first chart shows top 5 scores. The second chart shows averages and how many scores have the same value. You can toggle the chart by clicking its label.

Press SPACE to hover and try to avoid obstacles. You score when obstacles reach the other side. The game ends when you hit the obstacle and your score is then sent to the database.

It works best with 60hz.
