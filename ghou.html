Houston's Adventures Game

Unity

C#

Adobe Photoshop

Demo for a 2D platformer. It had a set of rooms that were randomly selected from the pool. Each room had a different type of environment, set of enemies and obstacles.The Player's goal was to get through the rooms as fast as possible and get the highest score. Defeated enemies dropped orbs that could be collected for points.

  • - Player controller: double jump, dash, groundcheck, hit knockback, diagonal aiming, health
  • - Unity new input system
  • - Camera controller with lookahead, boundaries and parallax
  • - 2D lights and particle systems
  • - Rigged player sprite with separate upper and lower body animations made in Unity
  • - Sprites made in Adobe Photoshop
  • - Wind and dissolve shader effects
